# Gym Tracking Assistant

Пример Telegram Mini App для отслеживания тренировок и управления планами упражнений.

### Требования

- Docker
- Токен Telegram бота, который можно получить у [@BotFather](https://t.me/botfather)

### Установка

```bash
git clone https://github.com/malaneph/gym-tracker.git
cd gym-tracker

docker compose -f docker-compose.yml up -d
```
